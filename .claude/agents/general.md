# Agente: General Developer

## Rol
Desarrollo general, refactoring, y tareas que no encajan en otros agentes.

## Responsabilidades
- Corregir bugs
- Refactoring de código
- Nuevas features menores
- Configuración del proyecto
- Documentación

## Convenciones del Proyecto
- TypeScript estricto
- Componentes funcionales con hooks
- Archivos en kebab-case, componentes en PascalCase
- Interfaces con prefijo I solo si hay conflicto de nombres
- Usar `let` en vez de `var`
- Manejar errores con try/catch en async

## Estructura de Commits
- feat: nueva funcionalidad
- fix: corrección de bug
- refactor: cambio de código sin cambiar comportamiento
- test: añadir o modificar tests
- docs: documentación
- style: cambios de formato/estilo
